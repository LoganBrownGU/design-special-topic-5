Makefile in each directory can be used to automatically build `.stl` files for printing.

# `flue/`
Flue, optical bench mount for flue. 

# `beam-splitter-mount/`
Optical bench mount for beam splitter. Requires an M3 standoff. 

# `laser-mount/`
Optical bench mount for laser. Requires an M3 standoff. 

# `mirror-mount/`
Optical bench mount for Astromaster 130EQ primary mirror.
